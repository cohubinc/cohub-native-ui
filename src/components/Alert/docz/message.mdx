# Message

<PropsTable of={Message} />

## Description

The cohub Message component is just a wrapper around the semantic-ui-react Message component.
Our version uses slightly different layout and colors. It also fades in and scrolls into view on render.

## Usage

<Playground>
  <div className="mb-2">
    <Message>Default Message</Message>

<div className="mb-2">
  <Message centerAlign>Message with centerAlign set to true</Message>
</div>
