# Icon

<PropsTable of={Icon} />

## Usage

### Access icons via static methods on Icon or using the name prop

<Playground>
  <div className="mb-3">
    <Icon.TagDollar />
  </div>

  <span className="mb-3">
    <Icon name="tagDollar" />
  </span>
</Playground>

### Change color, size, className, and or style

<Playground>
  <div className="flex">
    <Icon.Calculator />
    <Icon.Calculator
      size={44}
      color={Color.primary}
      className="mg-l"
      style={{ alignSelf: "flex-end" }}
    />
  </div>
</Playground>

### Availible Icons

<div className="border p-1">
  {iconNames.map(name => (
    <div className="mb-1" key={name}>
      <Icon name={name} size={24} />
      <span className="text-center">{name}</span>
    </div>
  ))}
</div>

### Colors can transition when changed

_Currently this logic is only implemented in the Circle icon_

<Playground>
  <DoczStateCtrl>
    {({ on }) => (
      <Icon.Circle color={on ? Color.primary : Color.darkBlack} size={24} />
    )}
  </DoczStateCtrl>
</Playground>

### Icons can be clickable

<Playground>
  <Icon name="circlePlus" size={24} onClick={() => console.log("clicked!")} />
</Playground>
